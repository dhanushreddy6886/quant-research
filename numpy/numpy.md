## About NumPy
NumPy (Numerical Python) is a fundamental Python library used for numerical computing. It provides the powerful `ndarray` data structure, which is faster and more memory-efficient than Python lists for mathematical operations.

## Topics I Learned

- Creating NumPy arrays (`np.array()`)
- Array attributes (`shape`, `size`, `ndim`, `dtype`)
- Indexing and slicing arrays
- Reshaping arrays
- Boolean masking and conditional selection
- Broadcasting
- Vectorized operations
- Mathematical and statistical functions
- Aggregation functions (`sum`, `mean`, `min`, `max`, `std`, etc.)
- Random number generation using `numpy.random`
- Basic linear algebra operations

## Key Takeaways

- NumPy arrays are faster and more efficient than Python lists.
- Vectorization eliminates the need for many explicit Python loops.
- Broadcasting allows operations between arrays of different shapes.
- NumPy provides built-in mathematical, statistical, and linear algebra functions that are essential for data analysis and quantitative finance.

## Why I Learned NumPy

NumPy forms the foundation of the Python data science ecosystem. It is widely used in libraries such as Pandas, SciPy, Matplotlib, Scikit-learn, and many quantitative finance applications. Learning NumPy is an important first step toward becoming a Quantitative Researcher.
